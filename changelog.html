LiveVersion Changelog



v2.9

19th September 2020

New Features
  • Added Automatic Migration Tool! This will automatically copy all data from old Sheets to the new one, making upgrades take just 30 seconds now.
  • New Calender reminders to run monthly data recording.
  • Readded support for CoinMarketCap crypto prices. All cryptos now supported!
  • Added new end of month progression prompt
  • "Sold Units" are now automatically calculated FIFO for every asset class.
  • Added MorningStar support to Managed Fund pricing
  • Added international currency support for Bank accounts. If you would like to see the balance of international accounts in your local/base currency, please unhide column D.
  • Improvements to ETF/Stock price fetching performance
  • Dividends for international assets will try and return a converted automatic "Price @ Dividend" price
  • Added new sector classification system
  • Emergency fund duration is now configurable in the SheetOptions tab
    • Bug fixes:
  • Fix for Managed Fund, Stock and ETF prices when sourced from Bloomberg.
  • Fix for dividends not being included in total return values.
  • Patch for crypto prices returning "ERROR".
  • Patch for investment autobalance referencing incorrect cell.
  • Fixes to Managed Fund allocation formulas & filtering.
  • Fix for Managed Funds not appearing on Net Worth tracker graph in Net Worth tab.
  • Fix for migration script copying over the current unwritten month.
  • Fix for automatic stock brokerage missing.
  • Fix for GBX currency erroring.
  • Cleaned up all Asset type graphs to be consistent.
  • Readjusted layout of dividends tab to allow for dividends from more than 15 companies at once.
  • Fixes to Gain (%) and values if original purchase price was $0.
  • Fixed bug where reinvested dividends were double counted as contributions.
  • Fixed issue with History sheet not migrating all data.
