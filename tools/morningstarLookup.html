How to use this new ID:
  1. Find your replacement Morningstar ID using the tool above
  2. Open the Managed Funds tab in your Sheet
  3. Click Edit → Find and Replace
  4. Next to the Search option, change this option from All Sheets → This Sheet
  5. Tick the Match entire Cell contents option
  6. In the Find box enter in the OLD ID. In the replace box enter in the NEW ID
  7. Click Replace All
